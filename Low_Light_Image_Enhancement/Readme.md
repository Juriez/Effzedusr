## Gamma Correction Demonstration

This module is **not part of the core Zero-Shot Image Enhancement project**. It has been included separately to demonstrate the effect of gamma correction on real-world low-light images.

Before running this module, ensure that the following folder structure is maintained:

```text
Low_Light_Image_Enhancement/
├── Input_images/
├── results/
│   └── outputs/
└── Low_Light_Image_Enhancement.py
```

### Usage Instructions

1. Place the low-light input images inside the `Input_images` directory.
2. Run `Low_Light_Image_Enhancement.py`.
3. The enhanced images will be saved in the `results/outputs` directory.
4. When specifying an image in the enhancement section, provide the **original image filename along with its extension** (e.g., `image1.jpg`, `sample.png`).
